## Interactive Registration Form

Ein kleines Frontend-Projekt mit HTML, CSS und JavaScript.

### Funktionen:
- Formularvalidierung (E-Mail & Passwort)
- Fehlermeldungen unter den Feldern
- Aktivieren/Deaktivieren des Buttons
- Animation bei Eingabefehlern

Ziel: Benutzerinteraktion und saubere Frontend-Logik.
